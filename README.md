# xp
## a command line agile development tool built in CoffeeScript and node.js


#  Usage
    xp [tool] [options]

## Getting Help

    xp
    xp -help
    xp --help


## Setting up Pair Programmers

    xp pair nr nk

## Getting Lines of Code

    xp loc [glob]
    xp loc .
    xp loc ../
    xp loc ./foo
